# [Code-directory](https://code-directory.com): Educational Platform for Programmers

## Description

Code-directory is a unique online platform providing an extensive catalog of questions and answers on popular programming languages. Our website facilitates effective learning and preparation for interviews, as well as the enhancement of programming skills.

## Features

### 1. Extensive Catalog of Questions and Answers

Code-directory offers a comprehensive catalog of questions and answers covering various programming languages. The questions span a wide range of topics, from basics to advanced concepts, enabling users to gain a complete and well-rounded learning experience.

### 2. Marking Functionality

The site allows users to mark questions as "learned" or "requiring review." This personalization feature helps in tailoring the learning experience and tracking progress in mastering different concepts. Filters based on markings make the process of selecting interesting questions easy and convenient.

### 3. [Adding Custom Questions](https://code-directory.com/add-question)

Users can contribute to the shared catalog by adding their own questions. This creates a personalized learning experience and expands the knowledge base for the entire community.

### 4. Interview Preparation

Code-directory is ideal for preparing for technical interviews. The marking functionality helps focus on questions that require additional attention, and the broad range of topics covers all aspects necessary for successful interview performance.

## Technology Stack

- **Node.js:**
  - Used to create an API that handles server requests. Implements functionality for user registration, login, and password encryption.

- **Next.js:**
  - Chosen as a universal wrapper for React, providing a more convenient and rapid development of the web application. Key advantages include easy state management, support for server-side rendering, and simplicity in configuration.

- **Nginx:**
  - Nginx serves as a web server, ensuring fast and efficient handling of requests. Used to add an SSL certificate, providing a secure connection and protecting user data.

- **Docker:**
  - Docker provides a unified container containing all dependencies and configurations, simplifying the deployment and scalability of the application. Used for creating an isolated application environment.

- **PostgreSQL:**
  - Utilized for storing questions with answers, paths to code examples, user data, and filters for questions. PostgreSQL is a powerful open-source relational database that ensures data integrity and supports complex queries. It plays a crucial role in managing the structured data of the platform.

- **DigitalOcean:**
  - DigitalOcean offers reliable computing resources and convenient management of virtual machines. A virtual machine was purchased on the DigitalOcean platform to host the web application.

- **GoDaddy:**
  - The domain name provides a convenient and memorable address for users, making access to the platform more user-friendly. The domain code-directory.com was purchased on the GoDaddy platform.
